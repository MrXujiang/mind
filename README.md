# flow-mind 一个基础的思维导图工具
An out-of-the-box visual mind editor.

## 主要特点

- **可视化搭建思维带图**
- **内置思维导图模版**
- **节点样式编辑器**
- **高质量导出**
- **AI问答模块**
- **模版和思维导图管理**

## 开发和构建工具

- **ESLint**：代码质量检查
- **PostCSS**：CSS 转换和优化
- **SWC**：快速的 JavaScript/TypeScript 编译器
- **Autoprefixer**：自动添加 CSS 前缀
